## Default Web Project Template (Vite + Tailwind CSS + HTML + JS)

### Quick start
- npm install
- npm run dev (http://localhost:5173)
- npm run build
- npm run preview

### Structure
- src/index.html
- src/assets/css/tailwind.css
- src/assets/js/main.js
- src/assets/images
- src/pages

### Build output
- dist/index.html
- dist/assets/css/styles.css
- dist/assets/js/bundle.js
- dist/assets/images
